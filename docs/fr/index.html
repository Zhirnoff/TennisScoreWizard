Aide rapide

Comment démarrer un match ?

Choisissez simple ou double, meilleur des 3 ou 5 sets, la règle de set, puis choisissez le premier serveur manuellement ou avec le tirage.

Comment compter les points ?

Touchez votre grande carte de score quand vous gagnez un point. Touchez la carte adverse quand l’adversaire gagne un point. L’app met à jour automatiquement points, jeux, sets et tie-breaks.

À quoi sert Menu ?

Pendant un match, Menu donne accès aux actions comme arrêter un match inachevé, ouvrir l’historique ou démarrer un nouveau match. Les matchs terminés sont détectés automatiquement quand le nombre de sets requis est gagné.

Comment les matchs terminés sont-ils enregistrés ?

Quand un joueur ou une équipe gagne le nombre de sets requis, l’app affiche automatiquement Match Finished et enregistre le résultat dans l’historique. Vous n’avez pas besoin de terminer le match manuellement.

Comment fonctionne Undo ?

Undo annule le dernier point saisi et restaure l’état précédent du match, y compris points, jeux, sets, serveur et position de service.

Comment fonctionne le marqueur de service ?

Quand vous servez, la balle indique le côté depuis lequel vous devez servir. Quand l’adversaire sert, elle indique son côté de service depuis votre point de vue sur la montre.

Comprendre l’historique des matchs

Les matchs enregistrés utilisent une notation tennis compacte pour rester lisibles sur Apple Watch.

Que signifient Win et Loss ?

Win signifie que vous avez gagné le match terminé aux sets. Loss signifie que l’adversaire a gagné le match terminé aux sets.

Que signifie Stopped ?

Stopped signifie que le match a été enregistré avant sa fin normale. Utilisez Menu pour arrêter un match non terminé.

Que sont S1, S2 et S3 ?

S1, S2 et S3 signifient Set 1, Set 2 et Set 3. Par exemple, S1 6–3 signifie que le premier set s’est terminé 6 jeux à 3.

Que signifie TB 7–3 ?

TB indique les points du tie-break dans un set gagné 7–6. Par exemple, 7–6 · TB 7–3 signifie que le set a été gagné 7–6 et que le tie-break s’est terminé 7–3.

Comment les matchs arrêtés sont-ils évalués ?

Si un match arrêté présente une avance claire en sets, l’historique indique qui menait aux sets. Si les sets sont à égalité, l’app compare les jeux puis les points de tie-break si disponibles.

Que devient un set inachevé ?

Pour les matchs arrêtés, le set en cours est enregistré s’il avait commencé, afin de conserver le contexte réel du match, par exemple S3 0–1.

FAQ

L’app collecte-t-elle des données ?

Non. L’historique des matchs est stocké localement sur votre appareil.

Puis-je l’utiliser en double ?

Oui. Sélectionnez Doubles dans la configuration avant de démarrer le match.

Puis-je jouer sans tie-break ?

Oui. Choisissez No Tie-break dans la configuration pour jouer un set avec avantage.

Puis-je supprimer l’historique des matchs ?

Oui. Les matchs enregistrés peuvent être supprimés depuis l’écran d’historique.

Confidentialité

L’historique des matchs est stocké localement sur votre appareil. L’app ne nécessite pas de compte et ne transmet, ne vend ni ne partage de données personnelles.
