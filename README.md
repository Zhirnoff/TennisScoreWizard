# Tennis Score Wizard Support

Tennis Score Wizard is an Apple Watch app for tracking tennis match scores directly from your wrist.

## Support

If you have questions, bug reports, or feedback, please open a GitHub issue:

https://github.com/Zhirnoff/TennisScoreWizard/issues

When reporting a problem, please include:
- Apple Watch model
- watchOS version
- App version
- Steps to reproduce the issue
- What you expected to happen
- What actually happened

## App Information

Tennis Score Wizard helps you track:
- Points, games, and sets
- Singles and doubles matches
- Best of 3 and Best of 5 formats
- Tie-break scoring
- First server selection
- Undo actions
- Local match history

## Privacy

Tennis Score Wizard does not require login and does not collect personal data.

Match history is stored locally on your Apple Watch.
